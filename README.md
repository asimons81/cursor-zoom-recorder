# Cursor Zoom Recorder

Cursor auto-zoom screen recorder for Windows (macOS later).

## MVP Features
- Capture: full screen, window, or region
- Zoom modes: always follow / click+drag / smart (slow-down)
- Output: MP4 + GIF
- Cursor highlight + click ripple
- Hotkeys for start/stop/pause

## Structure
- `app/` → desktop app (PySide6)
- `web/` → landing page (Vercel)
